# Анализ данных интернет-магазина

## О проекте

В этом проекте проводится анализ данных интернет-магазина. Основная цель - подготовить данные, найти основные тренды в продажах и сделать небольшой дашборд.

Проект состоит из двух частей: анализа в Jupyter Notebook и Streamlit-приложения. В ноутбуке выполнены очистка данных, расчёты и визуализации. В приложении можно загрузить CSV-файл, посмотреть очищенные данные, основные показатели и графики.

## Используемые инструменты

В проекте использовались:

- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit
- Jupyter Notebook

## Что было сделано

В ходе проекта были выполнены основные этапы анализа данных:

- загрузка CSV-файла с заказами;
- проверка пропущенных значений;
- проверка дубликатов;
- преобразование типов данных;
- обработка некорректных и пропущенных значений;
- расчёт выручки;
- анализ самых популярных товаров;
- анализ выручки по категориям;
- анализ продаж по месяцам;
- расчёт среднего чека;
- определение клиентов с наибольшим количеством заказов;
- построение графиков;
- создание Streamlit-приложения для просмотра результатов.

## Структура проекта

`online_store_analysis.ipynb` — Jupyter Notebook с подготовкой данных, анализом и графиками.
`app.py` — главная страница Streamlit-приложения.
`pages/visualisation_1.py` — отдельная страница с визуализациями.
`requirements.txt` — список библиотек, необходимых для запуска проекта.
`ecommerce_analytics_dataset.csv` — исходный CSV-файл с данными.
